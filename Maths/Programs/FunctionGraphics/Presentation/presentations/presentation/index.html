Python
Основа: ```py import numpy as np import matplotlib.pyplot as plt def graph (formula, x_range): x = np.array (x_range); y = eval (formula); plt.plot (x, y); plt.show (); ``` Начин на употреба: ```py graph('abs (x)', range(-100, 100)); # Графика на |x| graph('x', range(-100, 100)); # Графика на x graph('x * x', range(-100, 100)); # Графика на x ^ 2 graph('pow (x, 3)', range(-100, 100)); # Графика на x ^ 3 ```


Python
sin (x): ```py import matplotlib.pyplot as plt import numpy as np import math as math fs = 360 x = np.arange (fs) # [0, 360] y = [ np.sin (math.radians (i - fs)) for i in x] # множества от стойностите на sin за всяка стойност в x (в множеството [0, 360]) # sin в Python работи с радиани plt.plot (x, y) plt.show() plt.stem (x, y) plt.show() ```

Haskell


Haskell
Основа:
```hs import Graphics.EasyPlot main = do ```
Начин на употреба:
2D функция f (x) = sin(x) × cos(x)
```hs plot X11 $ Function2D [Title "Sine and Cosine"] [] (\x -> sin x * cos x) ```
Графика на sin
```hs plot X11 sin ```
Множеството от точки (sin(x), cos(x)), записано във файла img/plot.png като графика
```hs plot (PNG "img/plot.png") (sin . cos) ```
Графика на 2D функцията f(x) = 2cos (x)
```hs plot X11 $ Gnuplot2D [Color Blue] [] "2**cos(x)" ```
Графика на 3D функцията f(x,y) = x × y
```hs plot X11 "x*y" ```
Графика на точките (x, x3) за x ⊂ [-4, -3.9 ... 4] и 2D функцията f(x) = x2
```hs plot X11 [ Data2D [Title "Graph 1", Color Red] [] [(x, x ** 3) | x <- [-4,-3.9..4]], Function2D [Title "Function 2", Color Blue] [] (\x -> x ** 2) ] ```

GNU Octave


Octave
Начин на употреба:
2D функция f (x) = 5 × x
```m function result_of_f = f (x) result_of_f = 5 * x; endfunction ```
2D функция f (x) = sin(x) × cos(x)
```m function result_of_f = f (x) result_of_f = sin (x) .* cos(x); endfunction ```
Графика на sin
```m plot (sin); ``` ```m my_range = [-180:0.0001:180]; plot (my_range, f (my_range)); ```
Множеството от точки (sin(x), cos(x))
```m function ans = f (x) ans = cos (asin (x)); endfunction ```
Графика на 2D функцията f(x) = 2cos (x)
```m function ans = f (x) ans = 2 ^ cos (x); endfunction plot (my_range, f (my_range)); ```


Octave
Графика на 3D функцията f(x,y) = x × y
```m [xx, yy] = meshgrid (-1:0.002:1, 0:0.001:1); f = xx .* yy; surf (xx, yy, f) ```
Графика на f (x) = x3 за x ⊂ [-4, -3.9 ... 4] и 2D функцията f(x) = x2
```m function ans1 = f1 (x) ans1 = x .^ 3; endfunction function ans2 = f2 (x) ans2 = x .^ 2; endfunction x = [-(sqrt(25)):0.001:(sqrt(25))]; plot (x, f1 (x), "r-", x, f2 (x), "b--"); ```
Малко изненади ...
```m function asdf1 = f1 (x) x = x * 1.375; asdf1 = (2 * (x .* x .+ abs (x) - 6)) ./ (3 * (x .* x .+ abs (x) + 2)) - sqrt (25 - x .* x); endfunction function asdf2 = f2 (x) x = x * 1.375; asdf2 = (2 * (x .* x .+ abs (x) - 6)) ./ (3 * (x .* x .+ abs (x) + 2)) + sqrt (25 - x .* x); endfunction x = [-(sqrt(25)):0.001:(sqrt(25))]; plot (x, f1 (x), "r-", x, f2 (x), "r-"); ```
